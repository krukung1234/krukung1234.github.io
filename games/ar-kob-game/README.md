# เกม AR แม่ กบ

เว็บเกม AR สำหรับฝึกมาตราแม่ กบ แบบกบโผล่จากหลุม รอบละ 2 คำ ให้เด็กตีคำที่อยู่ในมาตราแม่ กบ

## วิธีเปิด

```powershell
.\serve.ps1
```

จากนั้นเข้า `http://127.0.0.1:4188/`

## วิธีเล่น

- เลือก `เริ่มแบบคลิก/แตะ`, `เริ่มแบบจีบนิ้ว AR` หรือ `เริ่มแบบไม่เปิดกล้อง`
- แต่ละรอบมีกบโผล่จากหลุม 2 ตัว พร้อมคำ 2 คำ
- ตีคำที่อยู่ในมาตราแม่ กบ เช่น คำที่สะกดด้วยเสียง บ ป พ ฟ ภ
- โหมดจีบนิ้ว AR ใช้ MediaPipe Hands ผ่าน CDN เพื่อตรวจจับโครงกระดูกมือจริง โดยกบยังโผล่จากหลุมให้เด็กเลือกตี
- ภาพกล้องและพิกัดมือถูกกลับด้านแกน X แบบส่องกระจก
- จีบนิ้วโป้งกับนิ้วชี้ให้วงแสงทับคำที่ต้องการ ระบบจึงจะนับเป็นการเลือก
- โครงมือเป็นสีม่วง-ฟ้า และเปลี่ยนเป็นสีทองเรืองแสงเมื่อจีบนิ้วสำเร็จ
- ตีถูกได้ 1 คะแนน ตีผิดได้ 0 คะแนน
- คำจะสุ่มแบบไม่ซ้ำจนกว่าคลังคำจะหมดก่อนจึงวนใหม่
