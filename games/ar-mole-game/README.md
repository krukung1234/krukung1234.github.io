# เกมตีตัวตุ่น แม่ ก กา AR

เว็บเกม AR แบบเปิดกล้องแล้วเล่นเกมตีตัวตุ่นซ้อนบนภาพจริง ใช้ได้ทั้งมือถือและเดสก์ท็อป

## วิธีเปิด

ตอนนี้ใน Codex เปิดไว้ให้ที่:

`http://127.0.0.1:4176/`

ถ้าต้องเปิดใหม่ภายหลัง ให้รันจาก PowerShell:

```powershell
.\serve.ps1
```

จากนั้นเข้า `http://127.0.0.1:4176/`

## วิธีเล่น

- เลือก `เริ่มแบบเปิดกล้อง AR` หรือ `เริ่มแบบไม่เปิดกล้อง`
- ถ้าเลือกเปิดกล้องแล้วเบราว์เซอร์ไม่อนุญาต ยังเล่นด้วยฉากการ์ตูนได้
- ตัวตุ่นขึ้นมาทีละ 2 ตัว และในแต่ละคู่มีคำแม่ ก กา 1 คำเสมอ
- ตีเฉพาะคำแม่ ก กา หรือคำที่ไม่มีตัวสะกด
- ถูกได้ 1 คะแนน ผิดได้ 0 คะแนน แล้วขึ้นคู่ใหม่ไปเรื่อย ๆ จนหมดเวลา
- คำจะสุ่มแบบไม่ซ้ำจนกว่าคลังคำจะหมดก่อนจึงวนใหม่
