วิธีเล่น

เกมนี้ฝึกคิดเป็นลำดับคำสั่งแบบ Coding เด็ก ๆ ต้องเลือกคำสั่งให้หนูเดินไปเก็บชีสและเข้าเส้นชัย

1. วางคำสั่งกดปุ่มเดินหน้า ถอยหลัง เลี้ยวซ้าย เลี้ยวขวา หรือเก็บชีส เพื่อใส่ลงคิวคำสั่ง
2. เก็บชีสหนูต้องยืนบนช่องชีส แล้วใช้คำสั่งเก็บชีส ชีสจึงจะหายและได้คะแนน
3. ทำคะแนนเข้าเส้นชัยให้สำเร็จ ใช้คำสั่งน้อยลงจะได้โบนัสเพิ่ม
4. ดูดาวดาวดูจากจำนวนคำสั่งที่ใช้ ถ้าใช้ไม่เกินเป้าหมายของด่านจะได้ 3 ดาว
