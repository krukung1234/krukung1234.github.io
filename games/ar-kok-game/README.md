# เกม AR แม่ กก

เว็บเกม AR สำหรับฝึกมาตราแม่ กก แบบกระต่ายโผล่จากหลุม ขึ้นคำทีละ 2 คำ ให้เด็กตีคำที่ถูกต้อง

## วิธีเปิด

```powershell
.\serve.ps1
```

จากนั้นเข้า `http://127.0.0.1:4177/`

## วิธีเล่น

- เลือก `เริ่มแบบเปิดกล้อง AR` หรือ `เริ่มแบบไม่เปิดกล้อง`
- แต่ละรอบมีกระต่ายโผล่จากหลุม 2 ตัว พร้อมคำ 2 คำ
- ตีคำที่อยู่ในมาตราแม่ กก เช่น คำที่สะกดด้วยเสียง ก ข ค ฆ
- ตีถูกได้ 1 คะแนน ตีผิดได้ 0 คะแนน
- คำจะสุ่มแบบไม่ซ้ำจนกว่าคลังคำจะหมดก่อนจึงวนใหม่
