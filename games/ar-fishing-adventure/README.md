# AR Fishing Adventure

เกมเป็นเว็บแบบ static สำหรับ GitHub Pages

## เปิดใช้งาน

1. อัปโหลดไฟล์ `index.html`, โฟลเดอร์ `assets/` และไฟล์นี้ขึ้น repository เดียวกัน
2. เปิด GitHub → Settings → Pages
3. เลือก Deploy from branch และเลือก branch `main` / โฟลเดอร์ `/root`
4. เปิดลิงก์ HTTPS ที่ GitHub Pages สร้างให้
5. เลือกด่าน → เลือกวิธีเล่น → กดเริ่มเกม

กล้องจะถูกขอหลังจากกด “เล่นด้วยกล้องและนิ้วมือ” เท่านั้น หากไม่อนุญาตหรือเปิดจาก `file://` ให้เลือก “เล่นด้วยการแตะหรือเมาส์” ได้ทันที

สำหรับตรวจสอบระบบมือชั่วคราว ให้ต่อท้าย URL ด้วย `?debugHands=1` เช่น `https://ชื่อผู้ใช้.github.io/ชื่อ-repo/?debugHands=1`
