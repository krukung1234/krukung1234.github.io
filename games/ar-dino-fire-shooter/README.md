# AR Dino Egg Cannon มาตราแม่ตัวสะกด

เกมการศึกษา AR สำหรับฝึกมาตราแม่ตัวสะกด ใช้ MediaPipe Hands ตรวจจับมือจริง และใช้ Selfie Segmentation เพื่อให้เห็นเฉพาะตัวคนอยู่ในฉากเกม

## วิธีเปิด

```powershell
.\serve.ps1
```

จากนั้นเปิด `http://127.0.0.1:4191/`

## วิธีเล่น

- กด `เริ่มเกม AR` แล้วอนุญาตกล้อง
- ชี้นิ้วเพื่อเล็ง Bubble คำตอบ
- จีบนิ้วโป้งกับนิ้วชี้เพื่อให้ไดโนเสาร์ใส่ไข่ลงกระบอกปืนแล้วยิง
- ยิงคำที่ตรงกับมาตราที่ถาม ถูกได้ +1 คะแนน
- ตอบถูก 3 ครั้งติดกันจะมีกล่องวิเศษด้านบน ให้โหม่งกล่องเพื่อรับ +5 หรือ +10 คะแนน

## ระบบ AR

- ใช้ MediaPipe Hands ผ่าน CDN
- ใช้ MediaPipe Selfie Segmentation ผ่าน CDN เพื่อแสดงเฉพาะตัวคน
- ภาพกล้องและพิกัดมือ mirror แกน X แบบส่องกระจก
- โครงมือปกติเป็นสีม่วง-ฟ้า
- เมื่อ Pinch จะเปลี่ยนเป็นสีทองเรืองแสง
