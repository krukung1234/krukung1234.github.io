Krukung Game Hub V9

สิ่งที่ปรับใหม่
- หน้าเว็บหลักสวยขึ้น
- หน้า Login ไม่มีข้อความรหัสเริ่มต้น
- ถ้ารูปปกโหลดไม่ได้ จะไม่โชว์ข้อความแตก ๆ

วิธีอัปเดต
1. แตก ZIP
2. คัดลอกไฟล์ทั้งหมดในโฟลเดอร์ krukung_v9 ไปวางทับใน krukung1234.github.io
3. เปิด GitHub Desktop
4. Commit
5. Push origin
6. เปิด https://krukung1234.github.io/?v=9 และ https://krukung1234.github.io/admin.html?v=9

วิธีเพิ่มเกม
1. เอาโฟลเดอร์เกมไปวางใน games เช่น games/ar-sudoku/index.html
2. Commit + Push ด้วย GitHub Desktop
3. เข้า Admin กรอกชื่อเกม รูปปก และชื่อโฟลเดอร์ เช่น ar-sudoku
4. กดบันทึก
