Krukung Game Hub V8 - Folder Mode

วิธีเพิ่มเกมแบบง่าย
1. เปิดโฟลเดอร์โปรเจกต์ krukung1234.github.io ในเครื่อง
2. เข้าโฟลเดอร์ games
3. สร้างโฟลเดอร์เกม เช่น ar-sudoku
4. ใส่ไฟล์เกมทั้งหมดไว้ในนั้น โดยต้องมี index.html
   ตัวอย่าง: games/ar-sudoku/index.html
5. เปิด GitHub Desktop > Commit > Push origin
6. เข้า admin.html
7. กดเลือกโฟลเดอร์เกม หรือพิมพ์ชื่อโฟลเดอร์ เช่น ar-sudoku
8. กดบันทึก

หมายเหตุ:
- ระบบนี้ไม่ใช้ Firebase Storage
- หน้า Admin จะบันทึกเฉพาะข้อมูลเกมลง Firestore
- ไฟล์เกมต้องวางในโฟลเดอร์ games และ Push ด้วย GitHub Desktop
